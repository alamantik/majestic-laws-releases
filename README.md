# Laws MEMO — поиск по законам и правилам Majestic RP (GTA 5 AltV / alt:V)

**Laws MEMO** — приложение-справочник для быстрого поиска по законодательной базе и правилам проекта Majestic RP на базе GTA 5 AltV (alt:V).
Подходит как **памятка для госника** и **памятка для админа/администратора**: помогает быстро находить нужные пункты правил, регламенты, наказания и процедуры по ключевым словам.

## Скачать (Releases)
Актуальные версии и обновления здесь:
https://github.com/alamantik/majestic-laws-releases/releases

## Кому полезно
- Госник (гос. структуры): памятка по обязанностям, процедурам, регламентам и базовым правилам.
- Админ (администрация сервера): памятка по правилам, наказаниям, регламентам и спорным пунктам.

## Возможности
- Быстрый поиск по законам и правилам Majestic RP.
- Поиск по ключевым словам и фразам (например: “памятка для госника”, “памятка для админа”, “правила”, “законы”, “наказания”, “регламент”).
- Удобный формат для ежедневного использования во время игры/модерации.
